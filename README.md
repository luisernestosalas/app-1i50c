# CRM con Clientes y Notas

Una aplicación web simple para gestión de relaciones con clientes (CRM) que permite administrar información de clientes y sus notas asociadas.

## 🚀 Demo

Puedes ver la aplicación en funcionamiento en: [https://app-1i50c-qmerc769v-luisernestosalas-2775s-projects.vercel.app](https://app-1i50c-qmerc769v-luisernestosalas-2775s-projects.vercel.app)

## ✨ Características

- **Gestión de Clientes**: Crear, editar y eliminar información de clientes
- **Sistema de Notas**: Añadir y gestionar notas para cada cliente
- **Interfaz Intuitiva**: Diseño simple y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador
- **Responsive**: Funciona en dispositivos móviles y desktop

## 🛠️ Tecnologías

- **HTML5**: Estructura y marcado semántico
- **CSS3**: Estilos y diseño responsive
- **JavaScript (Vanilla)**: Lógica de aplicación y manip
