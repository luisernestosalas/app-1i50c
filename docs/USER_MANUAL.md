# Manual de Usuario - CRM de Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar tus clientes y sus notas. Te ayuda a organizar la información de contacto y llevar un registro de las interacciones con cada cliente.

## Cómo acceder

**URL de acceso:** https://app-1i50c-qmerc769v-luisernestosalas-2775s-projects.vercel.app

Solo necesitas un navegador web. Funciona en computadora, tablet o celular.

## Funciones principales

### 1. Ver lista de clientes
- Al abrir la app, verás todos tus clientes
- Cada cliente muestra su nombre y datos básicos
- Los clientes se ordenan automáticamente

### 2. Agregar un nuevo cliente
1. Busca el botón **"Nuevo Cliente"** o **"+"**
2. Completa el formulario:
   - Nombre completo
   - Teléfono
   - Email
   - Empresa (opcional)
3. Haz clic en **"Guardar"**

### 3. Ver detalles de un cliente
1. Haz clic sobre el nombre del cliente
2. Se abrirá su ficha completa
3. Aquí verás:
   - Todos sus datos
   - Lista de notas guardadas
