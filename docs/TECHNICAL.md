## Información General

**Stack Tecnológico:** HTML + JavaScript (Frontend)

## Descripción del Sistema

Sistema CRM básico desarrollado con tecnologías web frontend que permite la gestión de clientes y notas asociadas. La aplicación utiliza un stack minimalista basado en HTML y JavaScript vanilla.

## Arquitectura

### Frontend
- **HTML**: Estructura y maquetación de la interfaz
- **JavaScript**: Lógica de negocio y manipulación del DOM
- **Almacenamiento**: LocalStorage del navegador (presumible)

### Hosting y Deploy
- **Plataforma**: Vercel
- **Tipo**: Static Site Deployment
- **CI/CD**: Integración automática con GitHub

## Funcionalidades Principales

- Gestión de clientes
- Sistema de notas
- Interfaz web responsiva
- Persistencia local de datos
